# Todo List X MVC with React

### 環境要求

Node.js 14.18.1
React 18.2.0


### 安裝方式

1.確認安裝完成Node.js與npm (npm init -y)

2.將專案 clone 到本地 

`git clone  https://github.com/LouisZhong/todo-list-react.git`
 
3.透過終端機進入資料夾(todo-list-react)，輸入：

`npm install`

4.安裝完畢後，繼續輸入：

`npm start`

5.瀏覽器自動進入到此網址： http://localhost:3000
